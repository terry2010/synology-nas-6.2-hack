# 使用方式:

          php index.php path=/volumeUSB1/usbshare/你的目录
          
          php index.php order=namelengthdesc path="/volumeUSB1/usbshare/你的目录"
          
           usb路径一般为 
           
                       /volumeUSB1/usbshare/你的目录
                       
                       /volumeUSB2/usbshare/目录名
                       
            volumeUSB后面的数字是第几个usb设备，这个需要自己尝试一下才知道
